INTRODUCTION                                    *lorem*

A very simple snippit plugin
Each snippit (or "lorem") is stored in it's own file

Lorems are searched for on the runtimepath
recursively under directories called "lorems"

Invoked using the "Lorem" command

Usage:
Lorem filename

The passed-in filename needs to be within 'lorems' dir within vim-lorem directory

Examples:
Lorem "lorem.fill"
Lorem "lorem.js"
Lorem "lorem.pl"
Lorem "lorem.txt"
Lorem "lorem.vader"
Lorem "mysnippet.txt"
